String 模仿基本类型的行为(独立操作值)，但本身是引用类型(A Class)
1. String默认值null, 对应Unicode码值，表示UTF-16编码方案
2. String是不可变值, 在创建之后不可更改, 所有值的更改都会重新创建一个对象
3. String能够存储字符长度只受到内存和Integer.MAX_VALUE值的大小限制
